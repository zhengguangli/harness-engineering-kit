# QUALITY_SCORE.md

按领域/分层追踪代码质量与架构一致性的评分,随时间观察差距是否在收敛还是扩大。由 `entropy-collector` agent 在周期性清扫时更新。

## 评分维度(示例,按项目调整)

- **结构一致性**:是否遵守 `ARCHITECTURE.md` 里定义的依赖方向
- **黄金原则遵循度**:`harness-golden-principles` 里编码的品味/惯用法规则的违反密度
- **测试覆盖**:关键路径的测试覆盖情况
- **文档新鲜度**:对应 `docs/design-docs/` 条目是否最近被校验过
